# Tomás Artes

Site de vendas do Tomás

Colocar o link aqui depois:

```text
/tomas-artes
│
├── index.html            # Página Inicial (Vitrine de Destaques + Sobre resumido)
│
├── pages/                # Pasta para as outras páginas
│   ├── galeria.html      # Catálogo completo (com filtros: Quadros, Camisas...)
│   ├── sobre.html        # A história do artista
│   └── contato.html      # Formulário ou links para redes sociais
│
├── css/
│   ├── global.css        # Variáveis de cor, reset, tipografia base
│   ├── home.css          # Estilos específicos da Home
│   ├── galeria.css       # Estilos do grid de produtos
│   └── components/       # Módulos reutilizáveis
│       ├── header.css    # Menu de navegação
│       ├── footer.css    # Rodapé
│       └── cards.css     # O estilo dos cartões de produto
│
├── js/
│   ├── main.js           # Lógica do menu mobile e scripts gerais
│   └── filtros.js        # (Futuro) Para filtrar "Quadros", "Canecas" na galeria
│
└── assets/               # Imagens do site
    ├── img/              # Fotos do site (logo, banners)
    └── produtos/         # As fotos das artes (aquelas que você mandou)
```
